# Retro Horror Prototype

This repository contains a small browser-based psychological horror prototype built with **Three.js**.

Features:

- Simple procedurally generated maze-like environment using cubes.
- First-person movement via the mouse and keyboard (W/A/S/D).
- Minimalist retro aesthetic.

## Running Locally

You can run a local web server in this directory and open `index.html` in a modern browser.

```
python -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).

## Controls

- `W` `A` `S` `D` to move
- Move the mouse to look around. You must click the screen first to lock the pointer.
- `Esc` releases pointer lock.

## Disclaimer

This is a simple experiment in procedural generation and psychological horror. It is not a full game and may contain flashing lights or unsettling atmosphere. Player discretion is advised.
